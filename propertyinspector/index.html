Launch Settings
When enabled, Steam will automatically enter Big Picture mode when launched.
Long Press Action
Choose what happens when you hold down the button for 2 seconds. "Close window" sends Alt+F4 to Steam (minimizes to tray if configured). "Exit Steam completely" gracefully shuts down Steam.
Button Behavior
Short Press:
• Steam not running → Launch Steam
• Steam running → Enter Big Picture mode
• In Big Picture → Exit to normal mode

Long Press (2 seconds):
• Configurable: Close Window (Alt+F4) or Exit Steam

Icon States:
• Greyed out = Steam not running
• Normal = Steam running (standard mode)
• Highlighted = Big Picture mode active
